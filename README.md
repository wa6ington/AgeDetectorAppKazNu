# Face Intelligence: AI-Анализатор Лиц

Интеллектуальная система для определения пола и возраста по фотографии с использованием глубокого обучения (Deep Learning) и OpenCV.

## Особенности
- **Высокая точность**: Использование средневзвешенного алгоритма для оценки точного возраста.
- **Современный интерфейс**: Интуитивно понятный GUI на базе Tkinter в темных тонах.
- **Модульная архитектура**: Чистый и организованный код для легкой поддержки.
- **Предобработка**: Адаптивное управление областью интереса (ROI) с динамическим падингом.

## Структура проекта
- `src/`: Исходный код (логика анализа и интерфейс).
- `assets/`: Предобученные нейронные сети и конфигурации.
- `samples/`: Примеры изображений для тестирования.
- `main.py`: Точка входа в приложение.

## Установка
1. Убедитесь, что у вас установлен Python 3.x.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование
Запустите приложение командой:
```bash
python main.py
```
Затем выберите фотографию, нажав кнопку «ЗАГРУЗИТЬ ИЗОБРАЖЕНИЕ».

## Технологии
- Python 3
- OpenCV (DNN module)
- Caffe Models (Age/Gender)
- Pillow (PIL)
- Tkinter
